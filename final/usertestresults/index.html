Final Project User Testing Results

First User Test

While filling out the input boxes, the user got stuck on the "your date" box. They were unsure if a name was supposed to be inputted there or not. Otherwise, they completed the test pretty quickly. They provided some suggestions on improvements:

1. Make the final output text larger

2. Make the characters in the background move with animation

3. Change the cursor to something else when clicking on the button

4. Include audio


Second User Test

The second user had some confusion when filling out the form. They were confused on what to put for "cuisine". Otherwise, it was pretty smooth sailing. A suggestion for improvement was breaking up the final text into two paragraphs so that it is less condensed.
